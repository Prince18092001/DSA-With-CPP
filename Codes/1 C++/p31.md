Q. Longest Prefix Suffix.
ans:- In this question we find longest prefix suffix (mean esma ham sab se longest string match ko output ma deta ha).

Example:-
    =>Firstly create prefix then Suffix.

           Prefix : a aa aaa aaaa aaaab aaaaba aaaabaa
           Suffix : a aa aaa baaa abaaa aabaaa aaabaaa
    
    => Then match each string and give then length of longest string match.

note:- 
      string match n-1 length tak because full length always match.

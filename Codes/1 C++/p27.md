Q. Roman to Integer (range [1-3999])
ans:- Here in this question change roman number to integer number and also here range is given [1-3999] (1 digit to 4 digit).

     Roman   Integer
       I  =>  1
       V  =>  5
       X  =>  10
       L  =>  50
       C  =>  100
       D  =>  500
       M  =>  1000

Example:
        Roman = CXL
        Integer = 100 - 10 + 50 => 140

steps to solve:
if next number greater then present then minus present.
otherwise add present (if next less than or equal to present case)
at last, end one number add (jo last me bacha hoga). 

code:-

// Roman to Integer range[1-3999]
#include<iostream>
using namespace  std;
int num(char c){
    if(c=='I')
       return 1;
    if(c=='V')
       return 5;
    if(c=='X')
       return 10;
    if(c=='L')
       return 50;
    if(c=='C')
       return 100;
    if(c=='D')
       return 500;
    if(c=='M')
       return 1000;

    return 0; // when not match then return 0
} // T.C. = O(1)
int main(){
  string s;
  cout<<"Enter a ROMAN number: ";
  getline(cin,s);
  
  int sum=0, index =0;
  while(index<s.size()-1){  // s.size-1 because next character se je compare kara ga
    if(num(s[index]) < num(s[index+1]))  // if next is greater then present then minus present other wise add
       sum -= num(s[index]);
    else
       sum += num(s[index]);
    index++;
  } // T.C. = O(N)
    sum += num(s[s.size()-1]);

    cout<<sum;

}

Time Complexity = O(N) + O(1) => O(N)
